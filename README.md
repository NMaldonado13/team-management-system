# Basketball Team Management System

An in progress full-stack web application built using Python and Flask that allows basketball teams to manage communication, organization, and performance data through role-based access for coaches and players.

## Motivation

I started this project to strengthen my udnerstanding of backend and full-stack development concepts beyond coursework. The focus is on learning how to design scalable web applications with authentication, permissions, database integration, and clean project structure.

## Tech used:

- Python
- Flask
- SQLite
- HTML/CSS
- Git and GitHub

## Current Features

- User authentication with session management
- Role-based access control (coach or player)
- Protected routes requiring login
- Shared layout with sidebar navigation
- Modular Flask routing and template structure
- SQLite database integration
- Database schema for announcements
